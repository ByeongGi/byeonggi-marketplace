# Initialize Project

Scaffold a new project using best-practice templates.

1. Ask the user what type of project they want to create (e.g., React, Python, Node.js).
2. Ask for the project name and location.
3. Use the appropriate tool (e.g., `npx create-react-app`, `python -m venv`) to create the project.
4. Set up initial configuration files (git, linting, etc.).
